Privacy Policy – Ruddr Time Tracker

Last updated: February 2026

Ruddr Time Tracker is a Chrome extension developed for internal use at Propellernet. This policy explains what data the extension accesses and how it is handled.

Data stored locally

The extension stores the following data in your browser's local storage (chrome.storage.local). This data never leaves your device except where explicitly stated below.

  • Member ID and name – your Ruddr workspace identity, used to filter time entries to your account.
  • Recent projects – the last 5 projects you logged time against, used to speed up entry creation.
  • Timer state – whether a timer is running and when it started.
  • Reminder settings – your end-of-day and periodic reminder preferences.
  • API key cache – a temporary copy of the Ruddr API key fetched from a secure internal endpoint, cached for up to 24 hours.

Network requests

The extension makes network requests to the following services:

  • ruddr.io – to read and write your time entries via the Ruddr API. Requests are authenticated with your organisation's API key.
  • Google Cloud Functions (europe-west2) – a Propellernet-managed endpoint used solely to deliver the Ruddr API key to the extension. No personal data is sent to this endpoint.

Cookie access

The extension reads browser cookies for ruddr.io to check whether you are currently logged in before making API calls. Cookies are read-only and are not modified or transmitted.

Notifications

The extension may display browser notifications to remind you to log your time. These are generated entirely on your device and do not involve any external service.

Data sharing

No personal data is shared with any third party. The only external services contacted are Ruddr (your time tracking platform) and a Propellernet-controlled Google Cloud endpoint.

Data retention

All locally stored data is held in your browser and is removed when you uninstall the extension. No data is retained on any server operated by this extension.

Contact

This extension is maintained by Propellernet. For questions, contact your internal analytics team.
